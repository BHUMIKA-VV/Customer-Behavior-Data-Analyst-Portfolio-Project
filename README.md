# Customer Behavior Data Analyst Portfolio Project

This repository demonstrates an end-to-end customer behavior analytics workflow using Python, SQL, and Power BI concepts.

## Project Overview

The project includes:
- Data preparation and cleaning in Python
- SQL query examples for customer behavior insights
- A notebook for exploratory data analysis

## Files Included

- `Customer_Shopping_Behavior_Analysis.ipynb` — notebook with data import, exploration, cleaning, and SQL connection steps
- `customer_behavior_sql_queries.sql` — SQL queries for customer analytics
- `customer_shopping_behavior.csv` — sample dataset

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/amlanmohanty1/customer-trends-data-analysis-SQL-Python-PowerBI.git
   cd customer-trends-data-analysis-SQL-Python-PowerBI
   ```
2. Open `Customer_Shopping_Behavior_Analysis.ipynb` in Jupyter Notebook or JupyterLab.
3. Review the data cleaning and exploration steps.
4. Run the notebook to load data into your SQL database if needed.
5. Use `customer_behavior_sql_queries.sql` to explore business questions with SQL.
